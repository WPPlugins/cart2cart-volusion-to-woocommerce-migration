== Description ==
**Supported WooCommerce versions: 1.1.x – 1.6.x, 2.x, 3.x** 

Cart2Cart: Volusion to WooCommerce Migration plugin gives an exclusive opportunity to transfer all database including products, orders and customers swiftly. The process is fully automated requiring no programming skills. Also, even during migration your Volusion store will be available online, what is going to avoid risks of traffic and sales decrease. Use a great chance to get a new online shop just in a few hours.

**The plugin offers 2 migration options**

1. Free Demo migration — lets you perform migration of products, orders, customers and categories in limited quantity (up to 10 entities each) for test purposes.
1. Full Migration — lets you perform UNLIMITED migration of products, orders, customers, categories, variants, attributes and other related entities. Full Migration price starts at $69 and may vary depending on the number of transferable entities and additional options chosen.

= Features =

* **No tech skills are needed** to perform transfer successfully as the whole procedure is automated. 
* **Costless Demo Migration** takes only 10-30 minutes.
* **Volusion store will be available online during transfer**.
* **Support Team** is ready to answer your questions via phone, email or live chat.

**Transfer the following data from Volusion to WooCommerce:**

* Products, product images, product variants;
* Categories, category images
* Customers, customer shipping address, customer billing address;
* Orders; order statuses.

**Please note:** Volusion template will not be migrated, because Cart2Cart transfers only data. 

Cart2Cart: Volusion to WooCommerce Migration plugin gives a unique possibility to move with no efforts and difficulties. First of all, you need to install and activate it and after that get your store connected via Connection Bridge. Then, click a button of start and just follow simple Wizard instructions. This plugin is absolutely free, but you need to pay for your data move which starts at $69. 
**Steps to take before migration:**

* Install WooCommerce store and make sure it can be reached online. 
* Check if you have FTP details for both shopping carts.  

== Installation ==

1. Get zip plugin file.
1. Unzip it to your computer.  
1. Upload unarchived file to wp-content/plugin directory.
1. Go to Admin -> Plugins, activate “Cart2Cart Volusion to WooCommerce Migration”.
1. Paste Volusion APi details. 
1. Connection Bridge for WooCommerce will be downloaded automatically.
1. Push Start Migration button and wait for a redirect to Cart2Cart page. Afterwards you can continue your Full Migration easily.
 
== Frequently Asked Questions ==
= Is there a possibility to migrate reviews to WooCommerce? =
Yes, Cart2Cart supports migration of reviews while moving to and from WooCommerce. Moreover, review ratings will be moved together with reviews.
= Can Cart2Cart migrate invoices? =
Cart2Cart doesn’t migrate invoices, but only orders. You target shopping cart will generate invoices automatically, if this function is available. In another case, you can generate invoices manually after migration is finished.
= When can I use my discount coupon? =
You will be able to use your discount coupon on a step of full migration confirmation. Simply put a tick in an appropriate checkbox and enter the promo code. 
= Can I pay for a service with Credit Card? =
Yes, you can pay with all major credit cards. Check 2Ceckout website for more info.
= Is it possible to migrate to the same domain on a different directory? =
Yes, it's possible to migrate within same domain using different folders.
